הגדרות XeLaTeX עבור VS Code

"latex-workshop.latex.tools": [
	{
		"name": "xelatex",
		"command": "xelatex",
		"args": [
			"-synctex=1",
			"-interaction=nonstopmode",
			"-file-line-error",
			"%DOC%"
		]
	}
],
"latex-workshop.latex.recipes": [
	{
		"name": "xelatex",
		"tools": [
			"xelatex"
		]
	}
],
"latex-workshop.latex.recipe.default": "xelatex"

📝 הוראות התקנה

1. פתח את VS Code
2. לחץ על Ctrl+, (או Cmd+, ב-Mac) לפתיחת ההגדרות
3. לחץ על סמל הקובץ בפינה הימנית העליונה לפתיחת settings.json
4. הדבק את הקוד שהעתקת בתוך הסוגריים המסולסלים
5. שמור את הקובץ (Ctrl+S)
הקוד הועתק בהצלחה! 🎉
