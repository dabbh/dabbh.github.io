üåç Global Configuration

Set up global settings that apply to all your projects

Creating Global .Run Files

Place a global configuration file at:

~/.vscode/global.Run # Linux/macOS
C:\Users\{username}\.vscode\global.Run # Windows

Example Global Configuration

# Global .Run configuration
[c]
compileFlags = -g -std=c99 -O2
safeMode = true

[python]
runCommand = python3 {filenameWithExt}

[java]
compileFlags = -cp .
