Installation Guide

Get the Run! extension up and running in 2 minutes

1

Open VS Code Extensions

Open Visual Studio Code and click on the Extensions icon in the sidebar, or use the keyboard shortcut:

Ctrl+Shift+X (Windows/Linux)
Cmd+Shift+X (Mac)
Alternative Method: Open Command Palette with Ctrl+Shift+P and type:
Extensions: Install Extensions
2

Search for Run! Extension

In the Extensions search bar, type "Run!" or "Danbh.run-a-code"

Search: Run!

Look for the extension with the ‚ñ∂Ô∏è icon and "Danbh" as the publisher.

3

Install the Extension

Click the "Install" button next to the Run! extension.

Direct Install: You can also click this link to install directly:
vscode:extension/Danbh.run-a-code
4

Verify Installation

After installation, you should see:

  • ‚úÖ A "Run" button in the status bar (bottom of VS Code)
  • ‚úÖ New commands in the Command Palette starting with "Run!:"
  • ‚úÖ Support for .Run configuration files
Note: The Run button will only appear when you have a supported file open (.c, .py, .java, etc.)
5

Test Your Installation

Let's test the installation with a simple Python script:

# Create a new file called test.py
print("Hello from Run! Extension!")
print("Installation successful! üéâ")

Save the file and click the "‚ñ∂Ô∏è Run" button in the status bar. You should see the output in the terminal!
